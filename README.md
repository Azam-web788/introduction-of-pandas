# 📊 Pandas Introduction

Pandas is a powerful **Python library** used for **data analysis and manipulation**. It helps you work with tables, CSV files, Excel data, and more.

## 🚀 Clone the Repository

```bash
git clone <repository-url>
cd <repository-folder>
```

## ▶️ Run the Project

Open your terminal and run:

```bash
jupyter notebook
```

Then open the notebook (`.ipynb`) file and start exploring Pandas.

## 📦 Install Pandas

```bash
pip install pandas
```
